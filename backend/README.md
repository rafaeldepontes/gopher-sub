# Gopher Sub Backend

[![language](https://img.shields.io/badge/language-Go-00ADD8?labelColor=2F2F2F)](https://go.dev/doc/)
[![version](https://img.shields.io/badge/version-1.25-9C27B0?labelColor=2F2F2F)](https://go.dev/doc/install)

This backend application handles the authentication process and when everything is okay, it also have to take care of the mail service.

## How to run

from the root folder:

```bash
cd ./backend
go mod tidy
go run main.go
```
